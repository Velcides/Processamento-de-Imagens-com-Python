## Descomplicando a criação de pacotes de processamento de imagens em Python


### O pacote "image_processing" é usado para:

Módulo "Processing":
- Correspondência de histograma;
- Similaridade estrutural;
- Redimensionar imagem;

Módulo "Utils":
- Ler imagem;
- Salvar imagem;
- Plotar imagem;
- Plotar gráfico;
- Plotar histograma;
